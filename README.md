## <img src="https://media.giphy.com/media/hpXFi6fjpVptmYjxLI/giphy.gif" width="35"/> What is Zephoryx?

> 🚀 **Zephoryx AI Lab** turns raw files into **strategic decisions** — not just charts.
>
> 📂 Upload any file → 🤖 AI classifies it → 📊 You get a **data story, forecast, simulation, and an action plan.**
